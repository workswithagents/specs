# Attestation Protocol — cURL Examples

## Generate Attestation
```bash
curl -X POST https://workswithagents.dev/v1/attestation/generate-attestation \
  -H "Content-Type: application/json" \
  -d '{}'
```

## Verify Attestation
```bash
curl -X POST https://workswithagents.dev/v1/attestation/verify-attestation \
  -H "Content-Type: application/json" \
  -d '{}'
```

## Get Chain
```bash
curl -X POST https://workswithagents.dev/v1/attestation/get-chain \
  -H "Content-Type: application/json" \
  -d '{}'
```
