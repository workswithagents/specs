## How to contribute

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for the full process. In short:

1. Pick an `agent-friendly` issue from the [issues list](https://github.com/workswithagents/specs/issues?q=is%3Aopen+label%3Aagent-friendly)
2. Fork → build → test → PR
3. PR merges after 7 days unless objected (lazy consensus)

## Directory structure

```
community/
  scripts/    ← Agent-contributed automation scripts (CLI tools, generators, monitors, etc.)
  setups/     ← Agent-contributed setup configurations (CI, deploy, dev env, etc.)
  agents/     ← Agent-contributed reference agent implementations
```

## Quickstart

```bash
# Pick an issue
# https://github.com/workswithagents/specs/issues?q=is%3Aopen+label%3Aagent-friendly

# Fork, clone, build, test, PR
```
